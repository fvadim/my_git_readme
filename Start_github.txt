my_git_readme

## настройка


* создать аккаунт на гитхабе :)
* установить гит на комп
* после установки надо ввести имя пользователя и мыло, лучше чтобы совпадало с гитхабом
> (или наоборот, сначала ставим локальный гит, а потом акк на гитхабе)


> после установки локального гита и создания аккаунта на гитхабе можно взять 
публичный ключ в файле <%USERNAME%>\.ssh\id_rsa.pub
(а если его еще нет, то лучше создать https://help.github.com/articles/generating-ssh-keys )
и добавить в аккаунте гитхаба тот самый публичный ключ, без него реально задалбывает вбивать логин и пароль


## работа с GIT 
1. git init
 -- это инициализация репозитория
2. git add . 
 -- добавление всего в папке в обработку гитом
3. git commit -m "..."
 -- здесь коммитим сразу с внесением комментария


## работа с GITHUB

1. Создать репозиторий на гитхабе

2. прописать локальному гиту УРЛ для гитхаба

	git remote add origin https://github.com/<username>/<repo_name>.git
>или
	git remote add origin git@github.com:<username>/<repo_name>.git
> (но у меня порт 22 закрыт, поэтому пробовал только первый вариант)

3. сделать git pull origin master (почему-то эту часть везде опускают)
	-- origin - название 
	-- master - название ветки

4. залить репозиторий на гитхаб

	push remote -u origin master


====================
это копия


5. сделать локальную копию репозитория
	git clone https://github.com/<username>/<repo_name>.git [<dirname>]
